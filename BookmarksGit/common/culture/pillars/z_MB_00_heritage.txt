#Cleaned these up a little bit. Heritages need to be broad to serve the gameplay purpose they have while CE and RoA did some un-needed new heritages such as RoA breaking up Turkic which, while neat, broke things like the Persia struggle. Heritages also referenced by a LOT of base game files (unlike languages) which means changing them can break events, scripts, decisions and etc. so it's best to keep heritages based on what the base game has if possible rather than add a ton of base game files to the mod
