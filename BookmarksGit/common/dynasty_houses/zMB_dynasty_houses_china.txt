# China Families

# AUH made it so families in China could not marry those with the same family name which is accurate...but the base game didn't do anything like combine the families and so, as far as the game was concerned, a Liu family with a dynasty id of 1 wasn't the same thing as a Liu family with a dynasty id of 2. To deal with that I've combined the many families into larger dynasties based on the family name. Sometimes a name is still correctly broken up since Wú is not the same as Wǔ. Many of these connections for the families are completely made up but it serves the purpose for gameplay reasons and coat of arms. Preferably CK3 would let you localize family names separate from the dynasty without impacting the character name so many of these dynasties could have their location in the localization.
